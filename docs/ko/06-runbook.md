# 운영 가이드 (Runbook)

설계가 아무리 잘 되어 있어도, 실제로 운영하다 보면 예상치 못한 상황이 생깁니다. 이 가이드는 일상적인 운영 작업과 비상 상황에서 어떻게 대응할지를 정리합니다.

::: warning
이 운영 가이드는 현재 초안 상태입니다. 실제 운영 환경에 적용하기 전에 법률 검토 및 내부 승인 절차를 거쳐야 합니다.
:::
