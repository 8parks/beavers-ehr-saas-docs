# 보안 컨트롤 카탈로그

> 시나리오 전반에 걸쳐 사용되는 AWS 네이티브 보안 컨트롤의 재사용 가능한 카탈로그입니다.

## 신원 및 접근 관리

### Amazon Cognito

**역할:** 사용자 인증 및 JWT 토큰 발급

**활용 이유:** AWS 네이티브 관리형 인증 서비스로, 멀티 테넌트 JWT 클레임을 지원합니다.

**보안 설정 항목:**
- MFA 활성화
- 강력한 암호 정책
- Custom Attribute를 통한 tenant_id 및 role 클레임 포함
- 토큰 만료 정책

**관련 위협:** T-005 손상된 토큰, T-002 과도한 권한 접근


### AWS IAM

**역할:** AWS 리소스에 대한 세밀한 접근 제어

**활용 이유:** 최소 권한 원칙을 적용하여 모든 AWS 서비스 접근을 제어합니다.

**보안 설정 항목:**
- 최소 권한 역할 설계
- 역할 기반 접근 제어
- 계정 간 IAM 역할


## 테넌트 격리

### Schema-per-Tenant (PostgreSQL)

**역할:** 데이터베이스 레이어에서 테넌트 데이터 격리

**활용 이유:** 완전 공유 테이블보다 강한 격리를 제공하면서 Database-per-Tenant보다 비용 효율적입니다.

**관련 ADR:** ADR-004


### Row-Level Security (PostgreSQL)

**역할:** 스키마 격리에 대한 심층 방어 추가

**활용 이유:** 서비스 레이어 버그 발생 시 데이터베이스 레이어에서 추가 격리를 제공합니다.

**관련 ADR:** ADR-004


## 데이터 보호

### AWS KMS

**역할:** 암호화 키 관리, PHI 암호화, 의료 기록 디지털 서명

**활용 이유:** AWS 관리형 키 인프라로 HSM 수준의 보안을 제공합니다.

**보안 설정 항목:**
- 키 정책으로 접근 제한
- 자동 키 교체 활성화
- 비대칭 키를 활용한 서명 (Sign/Verify API)

**관련 ADR:** ADR-006


### AWS Secrets Manager

**역할:** 데이터베이스 자격증명 등 비밀 정보 관리

**보안 설정 항목:**
- 자동 비밀 교체
- IAM 정책 기반 접근 제어
- VPC 엔드포인트를 통한 프라이빗 접근

**관련 위협:** T-007 DB 자격증명 유출


## 로그 및 감사

### AWS CloudTrail

**역할:** 모든 AWS API 호출 감사 로그

**보안 설정 항목:**
- 모든 리전 및 계정에서 활성화
- 로그 파일 검증 활성화
- S3 Object Lock 버킷으로 전송
- CloudWatch Logs 통합

**관련 위협:** T-003, T-010


### Amazon S3 Object Lock

**역할:** 감사 로그 변조 방지 (WORM)

**보안 설정 항목:**
- Compliance 모드로 보존 기간 설정
- 퍼블릭 접근 차단
- KMS 암호화


### Amazon Athena

**역할:** S3 저장 감사 로그 쿼리 및 분석

**보안 설정 항목:**
- 접근 제한된 IAM 역할
- 쿼리 결과 암호화


## 네트워크 보안

### AWS WAF

**역할:** 웹 애플리케이션 방화벽, SQL 인젝션, XSS 등 방어


### VPC 및 VPC 엔드포인트

**역할:** AWS 서비스 간 프라이빗 연결

**관련 ADR:** ADR-008


## 탐지 및 자동 대응

### AWS Config

**역할:** AWS 리소스 구성 상태 지속 평가

**보안 설정 항목:**
- 퍼블릭 S3 버킷 접근 규칙
- 암호화되지 않은 스토리지 규칙
- CloudTrail 비활성화 탐지
- 과도한 권한 보안 그룹 규칙

**관련 위협:** T-004, T-010


### AWS Security Hub

**역할:** 보안 이상 탐지 결과 집계 및 우선순위화


### Amazon EventBridge

**역할:** 보안 이벤트 라우팅 및 자동화 트리거


### AWS Systems Manager Automation

**역할:** 승인된 보안 대응 런북 실행

**보안 설정 항목:**
- 최소 권한 실행 역할
- 실행 이력 CloudTrail 기록

**관련 ADR:** ADR-007
